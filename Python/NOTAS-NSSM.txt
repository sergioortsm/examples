🧰 ¿Qué es NSSM?
NSSM es una utilidad gratuita que permite ejecutar cualquier aplicación o script como un servicio de Windows. Es más sencilla que programar un servicio en C# o usar pywin32.

✅ Pasos para instalar tu script Python como servicio con NSSM
1. 🔽 Descarga NSSM
Ve a: https://nssm.cc/download

Descarga la versión estable (ZIP).

Extrae el ZIP, por ejemplo en C:\Herramientas\nssm.

Asegúrate de usar la carpeta win64 o win32 según tu sistema operativo.

2. 📁 Prepara tu carpeta de trabajo
Supongamos que tienes tu carpeta de trabajo así:

makefile

Ventaja	Detalle
✅ No necesitas modificar tu script	NSSM gestiona todo lo que hace falta para correr en segundo plano
✅ Autoinicio	Se ejecuta al arrancar Windows automáticamente
✅ Fácil de quitar o modificar	No toca el Registro directamente
